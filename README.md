# 💬 Daavy's Chat

A lightweight and modular chat enhancement for **Foundry VTT v13**. It organizes the chat sidebar into independent tabs and adds several quality-of-life features to keep your chat clean and efficient.

---

## 🗂️ Core Features

- **📑 Tabbed Interface** — Seamlessly switch between **Chat**, **Game** (Rolls/Actions), and **Whispers**.
- **🧠 Smart Classification** — Automatically filters system messages, action cards, and dice rolls into the **Game** tab.
- **🔔 Notification Pips** — Visual indicators let you know when there are unread messages in inactive tabs.
- **🧹 Scoped Clear** — The trash can icon only clears messages from the currently active tab.
- **🚀 Global Clear** — Hold **SHIFT** while clicking the clear button to delete all messages across all tabs.

---

## ✨ Modular Enhancements (Player Configurable)

Every feature below can be toggled on/off in the module settings:

- **👤 Cleaner Chat** — Hide user avatars and names to save vertical space.
- **📏 Compact Chat** — Shrinks the message header for a more streamlined look.
- **📉 Collapsible Formula** — Hide roll formulas behind a clickable header to reduce clutter.
- **🛡️ Hide Damage Traits (PF2e)** — Removes redundant trait tags from damage rolls.
- **🏷️ Trait Filter (PF2e)** — Intelligently hides common/useless tags and limits the display to the top 3 traits (click to expand).
- **👁️ Hide Damage Buttons (PF2e)** — Automatically hides damage buttons after use (with a toggle to show them again).

---

## ⚙️ Settings & Customization

The settings menu is neatly organized into **General Settings** and **PF2e Only** groups. All settings are **Client-Scoped**, meaning each player can customize their chat experience without affecting others.

---

## 🛠️ Compatibility

- **Foundry VTT**: Version 13.
- **Systems**: Agnostic (Tabs work everywhere), with specialized logic and features for **PF2e**.
